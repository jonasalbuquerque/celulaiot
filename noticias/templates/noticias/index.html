Apresentação

O projeto de extensão denominado “Célula de Internet das Coisas”, da Universidade Federal do Ceará, propõe a formação de alunos no desenvolvimento de sistemas embarcados voltados para Internet das coisas, visando multiplicar esse conhecimento para outros alunos dos cursos de graduação de diferentes instituições de ensino superior do Ceará.

  • Saiba mais

Cursos

Oferecemos formação e capacitação aos alunos de instituições de ensino do Ceará para o mercado de tecnologia na área de desenvolvimento de sistemas embarcados voltados para Internet das Coisas.

  • Saiba mais

Projetos

Temos como foco o fomento à inovação no desenvolvimento de sistemas embarcados na região, desenvolvendo práticas de sistemas embarcados e implementando componentes de software para necessidades reais de instituições em Fortaleza.
